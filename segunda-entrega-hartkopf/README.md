# E-Commerce Web App

Proyecto de e-commerce desarrollado como Single Page Application (SPA) usando **React** y **Vite**, con gestión de productos y usuarios en **Firebase Firestore**.

---

## Tecnologías

- **Frontend:** React, Vite, CSS/Sass
- **Routing:** React Router
- **Base de datos:** Firebase Firestore
- **Almacenamiento de imágenes:** imgBB Storage
- **Estado global:** Context API

---

## Funcionalidades

- Listado de productos por categorías
- Detalle de producto con información completa
- Carrito de compras persistente
- Checkout con confirmación de orden
- Autenticación de usuarios (registro/login)
- Gestión de stock y productos desde Firestore

---

## Instalación y ejecución

1. Clonar el repositorio:

git clone https://github.com/eduhartkopf/NavegaLasRutas-Hartkopf

Instalar dependencias:

npm install

Ejecutar la app en modo desarrollo:

npm run dev

## **Notas**

Este proyecto es una demostración de SPA de e-commerce.

Se puede ampliar con pagos reales y mayor seguridad en reglas de Firestore.
